Contact Icon Plus enhances the contact management experience by adding custom icons to your contacts.

Makes it easy to distinguish between Individuals, Employees, Companies, Contacts.

And more by enabling a user preference to add Invoice Addresses, Delivery Addresses, Private Addresses and Other Addresses.

Key Features:

  • Custom icons for contact types
  • Improved contact categorization
  • Enhanced visual representation in the tree and kanban view
  • Use the custom widget in other views from the partner model
  • Group contacts by type for easier categorization and analysis
  • Advanced search filters for precise contact type queries

Icons Type:

Individual: Represents individual contacts.
Employee: Represents employee contacts.
Company: Represents company contacts.
Contact: Represents general contacts.
Invoice Address: Represents invoice addresses. Detail Mode
Delivery Address: Represents delivery addresses. Detail Mode
Private Address: Represents private addresses. Detail Mode
Other Address: Represents other addresses. Detail Mode
*Detail Mode: When enabled, provides a more detailed view of contact types.
